## Page & Function

> home페이지

- 운전자 모드와 관리자 모드 전환 가능

- 두 모드의 기능 차이는 없음
- 단지 테마 색상만 바뀜
- input과 하단 버튼을 누르면 검색 화면으로 넘어감

> 검색페이지

- 충전소의 주소를 입력하면(예 : 부산) 해당되는 충전소 결과 나옴

> 문제

- '부산'이 아닌 '서울'이나 다른 지역을 입력하면 버튼이 깨지는 현상 발생
- '대구'를 검색하면 '부산광역시 해운"대구"'처럼 의도하지 않은 결과 출력
